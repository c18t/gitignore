# gitignore

gitignore create .gitignore from downloaded templates from gitignore.io

## Install

With [fisherman]

```
fisher gitignore
```

## Usage

```fish
gitignore osx c++
```

## Options

* -u, --update:
    Update templates list.

* -o, --outout=file:
    Save created template to file.

* -h, --help:
    Show usage help.
